"Was This Article Helpful?" Plugin Documentation

Introduction

This user guide will assist you in installing, configuring, and using the "Was This Article Helpful?" plugin for your WordPress site. This plugin allows your readers to provide feedback on your articles by voting "Yes" or "No", displaying the results as a percentage.

Installation

To install the "Was This Article Helpful?" plugin, follow these steps:

  1. Download the plugin zip file from the WordPress plugin repository or the location provided.
  2. Log in to your WordPress dashboard.
  3. Navigate to Plugins > Add New.
  4. Click on the Upload Plugin button at the top of the page.
  5. Choose the downloaded zip file from your computer and click Install Now.
  6. After the installation is complete, click Activate Plugin.

Configuration

Once the plugin is activated, it automatically appends the voting buttons to the end of each article. No further configuration is needed for the basic functionality.

Usage

For Visitors

Visitors can vote on articles by clicking the "Yes" or "No" buttons at the end of the content. After they vote, they will see a thank you message and the updated voting percentages. Visitors are prevented from voting multiple times on the same article.

For Administrators

Administrators can view voting results for each article directly within the WordPress dashboard:

  1. Navigate to Posts and click on any article to edit.
  2. On the right-hand side, you will see a meta box titled "Voting Results" displaying the number of positive and negative votes.

Customization

If you want to customize the look and feel of the voting buttons, you can add custom CSS to your theme.

For example, to change the button colors, add the following CSS in Appearance > Customize > Additional CSS:

.vote-button {
    background-color: #your-color; /* Change the background color */
}

.vote-button.user-voted {
    border-color: #your-highlight-color; /* Change the border color for voted button */
}

Troubleshooting

If the voting buttons do not appear on your articles:

  • Make sure the plugin is activated.
  • Clear your website's cache if you're using a caching plugin.
  • Check if your theme's the_content filter is modified, which may prevent the buttons from being appended.
